## Installation

This project follows a structured setup process to ensure a smooth development environment across different operating systems. Follow the steps below to install dependencies, configure the application, and start the development server.

### 1. Prerequisites

Before installation, make sure you have the following installed on your system:

- **Node.js (v16 or above)** — Required for running JavaScript on the server, installing packages, and executing project scripts.
- **npm or yarn** — Package managers used to install project dependencies.
- **Git** — Used for cloning the repository and version control.
- **Code Editor (VS Code recommended)** — For writing and managing project files.

You can verify installations using:

```sh
node -v
npm -v
git --version
